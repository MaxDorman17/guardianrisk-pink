# Guardian Risk — Demo Site

A modern demo rebuild of the Guardian Risk Ltd website (business risk &amp; investigations consultancy).

Static site — no build step. Open `index.html`, or serve the folder:

```bash
python -m http.server 8791
```

## Stack
- Hand-written HTML / CSS / vanilla JS
- Google Fonts: Space Grotesk + Inter
- All graphics are SVG / CSS (no images beyond the logo)
